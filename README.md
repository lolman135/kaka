# Такс, можлива проблема незапуску — відсутність потрібних графічних бібліотек. 

Вінда дуже вередлива система, тому може по приколу не довстановити те, що треба.
Спробуйте прописати в консолі оцю команду:
```
python.exe -m pip install kivy[base] kivy_deps.sdl2 kivy_deps.glew
```

Після цього, перейдіть в папку вашого проекту (де лежать main.py та clicker.kv) і запустити код командою:
```
python.exe main.py
```
